# BMAD Fork: Notion + Neo4j

BMAD-METHOD fork with **Notion** as source of truth (agent names, personas, workflows) and **Neo4j** as the persistent memory layer. Planning: Jay → John → Winston. Execution: Bob → Brooks/Barry → Troy. Support: Allura, Paige, Dr. Quinn, Sarah Boone.

## Install in a new folder

You can use this fork in **any project folder**:

- **New project:** Clone this repo into a new folder and open that folder in Cursor.
- **Existing project:** Copy `_bmad/`, `.cursor/`, and `docs/` into your project root and configure `_bmad/core/config.yaml` and `_bmad/bmm/config.yaml`.

**Full setup and install guide:** [README-FORK-SETUP.md](README-FORK-SETUP.md)  
**Step-by-step install (clone or copy):** [docs/INSTALL-NEW-FOLDER.md](docs/INSTALL-NEW-FOLDER.md)
